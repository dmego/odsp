某企业产品运输配送成本优化

（分析报告）

背景简介
运输问题是运筹学中的一项重要的问题，运输成本在总物流成本中占很大比重。 目前我国大部分企业的自身产品产销整体运输成本相对来说比较高、时间较长、运输效率较低。这就需要我们对物流进行合理化， 在确保服务质量的同时，以最低的运输费用使产品由产地运至销地。 本案例选取某企业生产的产品由北京、上海等产地运往河北、海南等 销地的运费数据，以及各产地的产量数据；销地的销量数据。使用表上作业法来分析解决该问题。该问题的解决， 既可以加快物流的运输速度，及时使产品运达供应市场， 还能够加快企业资金的周转。

样本选择及数据来源
本案例中的数据来自网络，该数据为某企业2018年产品运输优化数据。该企业在北京、上海、广东等九个省份建有生产基地， 所以在这九个生产地都有相应的产品生产量。例如在北京生产基地2018年预计生产 7062.6 万台。同时该企业生产的产品会 销往河北、山西、四川等二十二个省份市区，其中每个省份市区需求量也就是销售量各不相同，例如其中该企业在河北的销量为 2240.2 万台。

可视化分析
根据表上作业法模型进行计算，并对计算结果进行相关类型的可视化分析。对应地图可视化和产销可视化，我们可以非常形象的看出 哪个产地需要运往哪个销地多少数量的产品才能让运输成本最低。在产销可视化图示中，我们可以看出，对于产地北京来说，它是最大的 产地，其生产量为 7062.6 万台，单达到最小运输成本时，北京的销售地为6 个，分别是河北销售 273.9 万台,内蒙古销售 1529.7 万台, 辽宁销售 1687.8 万台,吉林销售 746.46 万台,黑龙江销售 1310.5 万台,西藏销售 211.88 万台,甘肃销售 1302.36万台。
（图1：地图可视化分析图示）

从销地角度分析角度可以看出，哪个销售地依赖哪个产地最多，我们不难发现，黑龙江、内蒙古、辽宁、甘肃 这四个地方的产品完全依赖北京提供。而 安徽、江西、湖南三个地方的产品则完全依赖上海提供。 河北是所有销地里销量最高的一个省份。其大部分产品依赖于产地天津提供。
（图2：销地角度分析图示）

根据计算可得该案例最小运输成本为 1507525.74 万元，总剩余产量为 0 ，总剩余销量为 0 。 这说明该案例是产销平衡的运输问题，也就是所有产地的产量恰好等于所有销地的销量。
（图3：最小成本分析图示）

结论和建议
从可视化图表以及最后求出的最小成本可以得出如下结论，该案例的最小成本为 1507525.74 万元，总剩余产量为 0 ，总剩余销量为 0 。
北京、上海两大产地的销地最多，并且其销量也最多，如若想缓解这两地的压力，可以增加如贵州、重庆， 两地的产量，适当减少北京、上海 两地的产量。
